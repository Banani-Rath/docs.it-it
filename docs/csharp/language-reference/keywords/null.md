# null (Riferimenti per C#)
La parola chiave `null` è un'espressione letterale che rappresenta un riferimento nullo, ovvero che non fa riferimento ad alcun oggetto.  `null` è il valore predefinito delle variabili di tipo riferimento.  I tipi valore comuni non possono essere null.  Tuttavia, C\# 2.0 ha introdotto i tipi valore nullable.  Vedere [Tipi nullable](../../../csharp/programming-guide/nullable-types/index.md).
